# IPL Intelligence Engine

## Project Overview

The IPL Intelligence Engine is an end-to-end cricket analytics project developed using IPL ball-by-ball data from 2008–2025.

The objective of the project is to identify the factors that drive success in T20 cricket through advanced SQL analytics, statistical testing, interactive Power BI dashboards and machine learning.

## Expected Score Intelligence Engine

Inputs:

* Venue
* Powerplay Runs
* Powerplay Wickets

Output:

* Expected Final Score

Performance Above Expectation (PAE):

PAE = Actual Score − Expected Score

## Key Findings

* Death overs are the strongest differentiator of match outcomes.
* Wickets generate larger momentum shifts than boundaries.
* Championship teams exhibit balanced batting and bowling performance.
* Venue context significantly impacts scoring expectations.
* Match outcomes can be predicted with approximately 70–72% accuracy.
